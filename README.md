Getting Started

1) Prereqs
- Node.js 18+

2) Install deps
```bash
npm install
```

3) Environment
- Copy `.env.local.example` to `.env.local` and set `OPENAI_API_KEY`.

4) Dev
```bash
npm run dev
```

Open http://localhost:3000

Security
- Never expose your OpenAI API key in the browser or commit it to git.
